# Boolsche Algebra

## Gesetze

``1. Kommutativgesetz:``

A ∨ B = B ∨ A

A ∧ B = B ∧ A


``2. Assoziativgesetz:``

(A ∨ B) ∨ C = A ∨ (B ∨ C)

(A ∧ B) ∧ C = A ∧ (B ∧ C)


``3. Distributivgesetz:``

A ∨ (B ∧ C) = (A ∨ B) ∧ (A ∨ C)

A (B ∨ C) = (A ∧ B) ∨ (A ∧ C)


``4. Identitätsgesetz``

A ∨ 1 = 1

A ∧ 0 = 0
